# Trabajo Practico Integrador Programación 1

## Gestión de Datos de Países en Python

### Descripción del Proyecto

Este proyecto consiste en el desarrollo de una aplicación de consola en Python que permite gestionar información de países almacenada en un archivo CSV.

El sistema permite realizar operaciones de búsqueda, filtrado, ordenamiento, actualización de datos y generación de estadísticas, aplicando los conceptos fundamentales desarrollados en la materia Programación 1.

### Objetivo

Desarrollar una aplicación modular utilizando Python que permita administrar un conjunto de datos de países mediante el uso de listas, diccionarios, funciones, estructuras condicionales y repetitivas, manejo de archivo CSV, algoritmos de ordenamiento y cálculos estadísticos.

### Funcionalidades

- Cargar datos desde archivo CSV.
- Agregar países.
- Actualizar población y superficie.
- Buscar países por nombre. (coincidencia exacta/ parcial)
- Filtrar por continente.
- Filtrar por rango de población.
- Filtrar por rango de superficie.
- Ordenar por nombre.
- Ordenar por población.
- Ordenar por superficie.
- Mostrar estadísticas generales.

### Instrucciones de Ejecución

1. Clonar o descargar el repositorio.
2. Verificar que el archivo paises.csv se encuentre en la misma carpeta que el programa.
3. Abrir una terminal en el directorio del proyecto.
4. Ejecutar:

```
python paises.py
```

5. Seleccionar una opción del menú.

### Ejemplo de Uso

Opción: Buscar País por Coincidencia Parcial

Entrada:

```
arg
```

Salida: (todos los paises que cominzan con arg y sus datos)

```
argentina | Poblacion: 45376763 | Superficie: 2780400 | Continente: america
```
